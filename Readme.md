#  Object-Oriented Programming Course

## About The Project
This repository contains a collection of practical assignments and projects developed during my university course in Object-Oriented Programming (OOP). The primary goal of this codebase is to demonstrate a solid understanding of software design principles, logical architecture, and clean code practices.

## Technologies Used
* Java
* IntelliJ

## Core OOP Concepts Demonstrated
Throughout the source code, you will find practical implementations of fundamental object-oriented principles:

* **Encapsulation:** Protecting data integrity by restricting direct access to object states and utilizing appropriate access modifiers.
* **Inheritance:** Creating hierarchical class structures to establish relationships between objects and promote code reusability.
* **Polymorphism:** Utilizing method overriding and overloading to allow objects to be treated as instances of their parent class or interface.
* **Abstraction:** Hiding complex implementation details behind simplified interfaces and abstract classes.
